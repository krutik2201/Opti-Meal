# OptiMeal Architecture

OptiMeal is structured into 3 distinct roles:
1. **Student Application**: Mobile-first React interface for discovering food and locking time slots.
2. **Vendor Dashboard**: React-based control panel that provides real-time smart pacing insights to kitchen staff.
3. **Admin Dashboard**: Analytics control center.

## Key Technologies
- Frontend: React 18, React Router v6
- Backend: Flask, Python 3
- Data: LocalStorage synchronization (Mocked backend logic for real-time demonstration)
