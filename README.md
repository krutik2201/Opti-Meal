# 🔥 OptiMeal – Smart Queue & Kitchen Intelligence System

## 🚀 Problem

College canteens face:
* Long queues
* Uncertain wait times
* Kitchen overload

---

## 💡 Solution

OptiMeal is a:

👉 **Smart Queue & Kitchen Intelligence System**

That:
* Reduces waiting time
* Predicts rush
* Guides vendors in real time

---

## 🧩 Features

### 👨‍🎓 Student
* Order food with time slots
* See queue status
* Track order

---

### 👨‍🍳 Vendor
* Accept → Prepare → Ready system
* Kitchen load monitoring
* Smart alerts

---

### 🧑‍💼 Admin
* Dashboard overview
* System monitoring

---

## ⚙️ Tech Stack

* React
* Flask
* Tailwind CSS (CSS Variables styling)

---

## ▶️ How to Run

### Frontend:
```bash
cd frontend
npm install
npm start
```

### Backend:
```bash
cd backend
pip install -r requirements.txt
python app.py
```

---

## 🎯 Core Idea

This is NOT a food ordering app.

👉 It is a **Time & Queue Optimization System**
