# 🔥 OptiMeal – Smart Queue & Kitchen Intelligence System

## 🚀 Problem

College canteens suffer from:
* Long queues
* Uncertain wait times
* Kitchen overload

---

## 💡 Solution

OptiMeal is a:

👉 **Smart Queue & Kitchen Intelligence System**

That:
* Predicts rush
* Controls ordering slots
* Guides vendors in real-time

---

## 🧩 Key Features

### 👨‍🎓 Student
* Order with pickup time slots
* See queue status
* Track order

---

### 👨‍🍳 Vendor
* Accept → Prepare → Ready workflow
* Kitchen load indicator
* Smart alerts

---

### 🧑‍💼 Admin
* Dashboard overview
* System monitoring

---

## ⚙️ Tech Stack

* React (Frontend)
* Flask (Backend)
* Tailwind CSS

---

## ▶️ How to Run

### Frontend:
```bash
cd frontend
npm install
npm start
```

### Backend:
```bash
cd backend
pip install -r requirements.txt
python app.py
```

---

## 🎯 Core Idea

This is NOT a food ordering app.

👉 It is a **Time & Queue Optimization System**

---

## 👉 Demo Flow

1. Login as Student
2. Place order
3. Login as Vendor
4. Process order
